Stap 1

Surf op het internet over HTML en CSS. Wat kun je er mee creeren?

Stap 2

Download een texteditor naar keuze. Voor beginners is Notepad++ een goede editor, maar om meer inzicht te krijgen over je geschreven code is Visual Studio Code een aanrader. Ook kun je hierop plugins installeren zoals HTML snippets.

Stap 3

Zorg dat je een snelle browser hebt, dus geen Internet Explorer. Microsoft Edge Beta is echt een aanrader.

Stap 4

Creër een map, open de nieuwe map via Visual Studio Code.

Stap 5

Om te kunnen starten kun je even kijkje nemen op Youtube Tutorials of W3schools.

Stap 6

Voeg nu paar belangrijke opbouw tags toe zoals; < html >, < head >, < body >. Niet vergeten om deze tags te sluiten!

Stap 7

Om je stylesheet te kunnen gebruiken moet je het gaan linken met je html file. Dit kun je doen met < link > tag waarin je angeeft dat het een stylesheet is.

Stap 8

Nu kun je elements toevoegen om inhoud toe te voegen in je index file. Zoals voor titels < h1 >, paragrafen < p >.

Stap 9

Tijdens of nadat je je code hebt geschreven kun je in de stylesheet per element of per sectie vormgeven. Kijk hiervoor naar tutorials.

Stap 10

Opslaan en uploaden/pushen in github.

Tip

Tijdens het maken je pagina kun je je code openen met je browser om je voortgang te zien. Zo krijg je een duidelijke beeld wat een tag of element voor zorgt. Zorg ook dat je code overzichtelijk is in de editor. Dus dat elementen tussen tags met wat afstand beginnen zie: Clean HTML
